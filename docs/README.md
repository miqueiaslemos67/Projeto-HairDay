# Documentacao do Projeto

Esta pasta centraliza os materiais de apoio criados durante o desenvolvimento em aula.

## Arquivos disponiveis

- `docs/documentacao-projeto.md`: visao geral do projeto, arquitetura e organizacao das pastas.
- `docs/proximos-agendamentos.md`: explica a separacao entre `Agendamentos do Dia` e `Proximos Agendamentos`.
- `docs/form/ajustes-agendamento.md`: descreve ajustes feitos no fluxo de troca de data e envio do formulario.
- `docs/form/correcao-agendamento.md`: registra a correcao para evitar agendamentos duplicados no mesmo dia e horario.

## Objetivo

Esses arquivos complementam o codigo-fonte com contexto de estudo, decisoes tomadas em aula e explicacoes sobre correcoes aplicadas na interface e na regra de negocio.
